* **描述：**  

  一起来看小姐姐，咦，我的小姐姐呢

* **flag：** flag{easy_misc}

* **Writeup：** 

  打开获得两个文件，使用01editor或wireshark打开图片，在图片中间发现压缩包，分离，发现有密码，但是zip文件数据区加密位为0000，zip文件目录区全局标志位为0900，故为zip伪加密，将目录区全局标志改为0000，解压获得第一个小姐姐；另外一个zip在末尾发现50 4b等zip头文件代码，转换为正常的zip，解压获得另一个小姐姐，使用盲水印解密获得flag。
